**Cartopia – Full-Stack Grocery Shopping Platform**  

**Description**  
A responsive, scalable online grocery marketplace built with the MERN stack, enabling seamless interactions between shoppers and sellers. Features include secure user authentication, dynamic product management, cloud-based image handling, and comprehensive order processing workflows.  

**Key Achievements**  
- Developed front-end with **React**, implementing responsive design and accessibility best practices for cross-device usability.  
- Built RESTful APIs in **Node.js**/**Express** to handle product catalogs, user sessions, orders, and inventory management.  
- Modeled and optimized **MongoDB** schemas for products, users, orders, and addresses; utilized indexing to accelerate query performance.  
- Implemented JWT-based authentication and **bcrypt** password hashing middleware for secure user login, registration, and role-based access control.  
- Created dynamic product categorization (fresh produce, packaged goods, daily essentials), best-seller highlights, and detailed product views to drive engagement.  
- Integrated cloud storage (AWS S3/Cloudinary) for product image uploads, automated resizing, and CDN delivery to ensure fast load times.  
- Developed seller dashboard for real-time product listing, order tracking, and inventory updates via secure API endpoints.  
- Managed user address CRUD operations and end-to-end order processing, including status updates and order history.  
- Leveraged Git/GitHub for version control; participated in Agile sprints, conducted code reviews, and set up CI/CD pipelines for automated testing and deployment.  
- Ensured maintainable code through modular architecture, middleware patterns, and comprehensive unit and integration tests.  

**Technologies:** React, Node.js, Express, MongoDB, JWT, bcrypt, AWS S3/Cloudinary, RESTful APIs, Git, Agile, CI/CD.
